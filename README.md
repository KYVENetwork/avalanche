## About

The KYVE + Avalanche node allows you to bridge any data stream from an Avalanche chain onto Arweave.

## Usage

### Run the node locally

1. Clone the repo.
2. Copy or create a new Arweave keyfile into the repo and rename it to `arweave.json`.
3. To run the node, execute `yarn start`.
